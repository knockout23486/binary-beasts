# 🛡️ Binary Beasts: AI Phishing & Social Engineering Detector

A smart cybersecurity tool that detects phishing attacks and social engineering scams before they harm users.

Binary Beasts is an AI-powered threat detection system designed for the Hack2Skill Hackathon. Instead of relying only on outdated blocklists, it uses semantic AI analysis and OCR technology to detect hidden threats across different communication channels.

## ✨ Key Features

🧠 **AI Semantic Analysis (Gemini 2.5 Flash)**  
Analyzes message intent, tone, and psychological manipulation techniques used in phishing.

📸 **OCR Threat Detection (Tesseract.js)**  
Scans screenshots from WhatsApp, Instagram, SMS, and emails to detect hidden phishing messages.

🔗 **Smart Domain Reputation Analysis**  
Identifies suspicious domains, risky TLDs (.xyz, .top, etc.), and zero-day phishing links.

📊 **Real-time Threat Scoring**  
Generates a **0–100 risk score** with clear explanations and highlighted red flags.

## 🛡️ Problem Statement

Build a dynamic and intelligent system capable of identifying sophisticated digital scams across multiple platforms and communication channels.

The solution must go beyond traditional blocklists and detect emerging phishing threats using intelligent analysis.

---

## 📌 How It Works

1. User enters a message or uploads a screenshot.
2. OCR extracts text if the input is an image.
3. AI analyzes the message using semantic understanding.
4. The system evaluates domain reputation and phishing indicators.
5. A **threat score and explanation** is generated.
